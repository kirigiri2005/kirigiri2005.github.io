mikumo的咖啡日记

记录每一包豆子的来处，与每一杯咖啡的味道。

🛡️ 离线优先 📱 数据自有 🔓 开源免费
📥 下载 APK ⭐ GitHub 仓库
核心功能
🫘

咖啡豆库存管理

记录产国、产区、庄园、豆种、处理法、烘焙度与风味。克价自动计算，库存实时扣减。

📝

每日冲煮记录

记录克数、方式、水温、研磨度、风味笔记。选豆时实时显示剩余克数，库存不足自动拦截。

📊

统计图表

消耗占比饼图、30 天趋势折线图、冲煮方式柱状图。暖棕配色，自动深色模式。

💾

数据备份

JSON 导出备份 + 导入恢复，导入前自动备份防丢失。冲煮记录可导出 CSV。

🔒

隐私优先

SQLite 本地存储，无需注册、不上传服务器。你的数据永远在你手里。

🌙

赏味提醒

设置最佳赏味天数，卡片自动提醒。低库存警告，豆子用完自动隐藏。
