关于这个小站

这里是我的咖啡生活记录空间。从入门手冲到探索意式，从第一包精品豆到建立起自己的冲煮体系。 每一杯咖啡都是一次实验，每一包豆子都有一段故事。

📱 mikumo的咖啡日记 App

一个完全运行在手机本地、无服务器的咖啡豆库存管理与冲煮记录 Android 应用。 离线优先，数据自有，记录每一包豆子的来处与每一杯咖啡的味道。

📦 豆库管理

记录咖啡豆的产地、烘焙度、处理法、购买日期等信息

☕ 冲煮记录

记录每次冲煮的参数：粉量、水温、时间、风味描述

📊 数据统计

消耗趋势、豆子评分分布、冲煮参数对比

🔒 离线优先

数据完全存储在本地 SQLite，无需网络，保护隐私

📂 查看项目源码 React Native · Expo SDK 52 · SQLite · TypeScript
